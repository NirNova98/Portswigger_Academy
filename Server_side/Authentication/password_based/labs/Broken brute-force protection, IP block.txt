Vulnerability:
	The server reset the failed login cooldown after a successfully login.
	Allowing an attacker to bruteforce a known/valid username by logging in every 1-2 failed logins.
	
Exploit:
	Bruteforce the password over the given username one or two time at a time.
	Login to the given username:password (wiener).
	
Example:
	None
	
Prevention:
	- Implement an mfa.
	- Track the IP & Counts of failed requests.
